С началом небольшого изучения питона, мне сложно что-то делать на java.  
Вернее вести и запоминать их параллельно.  
В данном случае пришлось покопаться в синтаксисе питона, снова уяснить его особенности кортежа, распаковки, применения функций.  
В общем виде предложил бы следующее решение
```python
from functools import reduce

def second_max(numbers: list[int]) -> int:
  if len < 2:
     raise ValueError("Нужно минимум два числа")

  first, second = numbers[0], numbers[1]

  initial = (
        max(first, second),
        min(first, second)
    )

  def step(acc: tuple[int, int], value : int) -> tuple[int, int]:
    maximum, second_maximum = acc

    return (
      max(maximum, value),
      max(second_maximum, min(maximum, value))
    )

  return reduce(step, numbers[2:], initial)[1]
```

Наверное было бы не совсем честно для себя не попробовать написать всё тоже самое на java, раз здесь даются для него примеры
