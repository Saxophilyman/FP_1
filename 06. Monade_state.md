Ценность монад становится яснее и понятнее. Но воспринимается всё равно немного сложно.  
Концепция из крайнего урока мне была понятнее, чем промежуточные её этапы, всё-так с готовым её применением уже довелось на правктике работать.  
Пример с планированием финансовых действий - очень классический, но наглядный.  

Придумать какой-нибудь пример вызывает затруднения. Что-то серьёзное - слишком сложно пока что, что-то тривиальное - тоже не хочется.
Попробую предложить такой вариант. К удивлению - многие элеементы мы уже проходили или даже применяли.


```python
from pymonad.state import State


def append_text(value):
    def operation(text):
        def computation(history):
            new_text = text + value
            new_history = history + [text]

            return new_text, new_history

        return State(computation)

    return operation


def replace_text(old, new):
    def operation(text):
        def computation(history):
            new_text = text.replace(old, new)
            new_history = history + [text]

            return new_text, new_history

        return State(computation)

    return operation


def uppercase(text):
    def computation(history):
        new_text = text.upper()
        new_history = history + [text]

        return new_text, new_history

    return State(computation)


program = (
    State.insert("Hello")
        .then(append_text(", world"))
        .then(replace_text("world", "Python"))
        .then(uppercase)
)

result, history = program.run([])

print(result)
print(history)

```
